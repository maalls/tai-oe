🧾 Identification du produit

Champs permettant d’identifier de manière unique et normalisée un article.

marque : marque ou fabricant du produit

refciale : référence commerciale officielle du fabricant

refinfor : référence informative ou alternative (ancienne, interne, équivalent)

gtin : code GTIN / EAN permettant l’identification internationale

ctyp : type d’article (produit, service, accessoire, etc.)

🧩 Gamme et fonction

Champs décrivant la logique produit et son usage technique.

gamme : gamme ou série commerciale du produit

fonction : fonction technique principale de l’article

🏷️ Désignation / libellés

Textes descriptifs de longueurs différentes selon l’usage.

libelle40 : libellé court (facture, bon de livraison)

libelle80 : libellé standard (devis, ERP)

libelle240 : description longue (catalogue, site web)

💰 Tarification et fiscalité

Informations de prix et de TVA.

datetarif : date d’application du tarif

tarif : prix catalogue HT

tarifd : prix dégressif ou remisé

qt : quantité à laquelle le prix s’applique

tva : taux de TVA applicable

📦 Unités et conditionnement

Informations liées aux unités de vente, de commande et aux contraintes de quantité.

ub : unité de base (pièce, mètre, kg, etc.)

uc : unité de commande (boîte, carton…)

uccoef : coefficient entre unité de commande et unité de base

qmc : quantité minimale de commande

qmv : quantité minimale de vente

qmvt : quantité multiple de vente obligatoire

mul : multiplicateur logistique

qpki : quantité par kit si vendu en kit

🚚 Logistique et disponibilité

Champs liés au stock, aux délais et au cycle de vie.

st : stock disponible

sta : statut de l’article (actif, bloqué, supprimé)

delai : délai de livraison en jours

dateprev : date prévue de réapprovisionnement

edi : article commandable via EDI

dug : date ultime de gestion (fin de vie)

🌍 Réglementation et douane

Informations réglementaires et export.

douane : code douanier (TARIC / SH)

📊 Marketing

Axes marketing et classifications internes non techniques.

mkt1 : code marketing niveau 1

mkt1l : libellé marketing niveau 1

mkt2 : code marketing niveau 2

mkt2l : libellé marketing niveau 2

mkt3 : code marketing niveau 3

mkt3l : libellé marketing niveau 3

mkt4 : code marketing niveau 4

mkt4l : libellé marketing niveau 4

mkt5 : code marketing niveau 5

mkt5l : libellé marketing niveau 5

🗂️ Familles produits

Arborescence de classement produit (catégorisation).

fam1 : famille produit niveau 1

fam1l : libellé famille niveau 1

fam2 : famille produit niveau 2

fam2l : libellé famille niveau 2

fam3 : famille produit niveau 3

fam3l : libellé famille niveau 3
